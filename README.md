# No Longer Supported
This iteration is no longer supported. All development efforts are going towards a [later iteration](https://github.com/nyrex-io).

# Niche65
The Niche65 is an early development iteration of the nRF52840 based split wireless Bluetooth 65% mechanical keyboard. 

With the [Adafruit nRF52 Bootloader](https://github.com/adafruit/adafruit_nrf52_bootloader), the board can be made to run any UF2 based firmware. Common compatible firmware includes: ZMK, RMK, and KMK.
